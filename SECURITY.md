# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in this project, please report it by sending an email to **security[at]zahorniak.com**.

Please include:
- A description of the vulnerability
- Steps to reproduce the issue
- Possible impact
- Any suggested fixes (optional)

We will acknowledge receipt of your vulnerability report within 48 hours and will send you progress updates every 5 days. After the initial reply to your report, we will endeavor to keep you informed of the progress towards a fix and full announcement.

We ask that you do not disclose the issue publicly until we have had a chance to address it.

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| Latest  | :white_check_mark: |
| < Latest| :x:                |

Only the latest version of this pre-commit hook receives security updates.
